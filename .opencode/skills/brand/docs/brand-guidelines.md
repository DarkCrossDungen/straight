# STRAIGHT Brand Guidelines

## Brand Identity
- **Name:** STRAIGHT
- **Tagline:** Dictate. Straight.
- **Voice:** Direct, confident, no filler. Brutally simple.
- **Personality:** Industrial, editorial, high-contrast, zero frills.

## Visual Identity

### Design Philosophy
Brutalist monochrome — pure black/white foundation, red as power, yellow as accent. Zero radius. Hard shadows. Typography as texture.

### Color Palette
- **Background:** Pure White (#FFFFFF) / Pure Black (#000000)
- **Foreground:** Pure Black (#000000) / Pure White (#FFFFFF)
- **Primary (Red):** #DB0000 (light) / #FF0000 (dark)
- **Accent (Yellow):** #FFD700 (light) / #FFCC00 (dark)
- **Muted:** #F5F5F5 (light) / #1F1F1F (dark)
- **Muted Foreground:** #666666 (light) / #B3B3B3 (dark)
- **Border:** Pure Black (#000000) / Pure White (#FFFFFF)

### Typography
- **UI (Sans):** Inter, Helvetica Neue, Arial, sans-serif
- **Technical (Mono):** SF Mono, Courier New, monospace
- **Reading (Serif):** Georgia, serif
- **Letter Spacing:** -0.02em (tight)

### Spacing
- Base unit: 0.25rem (4dp)
- 4/8/12/16/20/24/32/48/64dp rhythm

### Effects
- **Border Radius:** 0px (every element)
- **Shadows:** 3px offset X/Y, 0px blur, 100% opacity — hard drop shadows only
- **Transitions:** Instant or 150ms max — no smooth/eased transitions
- **Dividers:** Full-bleed, 1px or 4px, pure black/white

## Usage Rules
1. Never round any corner — radius is always 0
2. Shadows are always hard (0 blur) with 3px offset
3. Use full-bleed horizontal rules as section dividers
4. No gradients, no glass effects, no rounded containers
5. Red is for primary actions and active states
6. Yellow is for highlights and secondary accents
7. Text is always max contrast against its background
8. Icons are always the same stroke weight — sharp, minimal
